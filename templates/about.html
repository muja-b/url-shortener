About This URL Shortener

Features
  • Shorten long URLs to easy-to-share links
  • Copy short URLs to clipboard
  • RESTful API endpoints
API Endpoints
GET /api/url - Get all shortened URLs
POST /api/url - Create a new shortened URL
GET /api/url/<short_url> - Get a specific shortened URL
DELETE /api/url/<short_url> - Delete a shortened URL
Getting Started
  1. Install dependencies: pip install -r requirements.txt
  2. Run the application: python app.py
  3. Open your browser to http://localhost:5000
  4. Start shortening your URLs!
